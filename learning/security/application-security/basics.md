# Application Security Basics

Principles and paradigms protecting application runtimes from vulnerabilities.

## Core Concepts

- **Least Privilege**: Restricting service executions to minimal required resources.
- **Defense in Depth**: Implementing multi-layered controls across networks, databases, and code.

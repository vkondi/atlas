# XSS & CSRF Remediations

Protecting clients from session hijacking and cross-site scripting injections.

## Core Concepts

- **Cross-Site Scripting (XSS)**: Sanitizing dynamic inputs and configuring Content Security Policies (CSP).
- **CSRF Defenses**: Configuring anti-CSRF headers and SameSite cookie policies.

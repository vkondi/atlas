# Mocking Paradigms

Replacing external dependencies with predictable test doubles.

## Core Concepts

- **Test Doubles**: Mocks, Spies, Stubs, and Fakes configurations.
- **Network Interception**: Intercepting network API calls using tools like MSW (Mock Service Worker).

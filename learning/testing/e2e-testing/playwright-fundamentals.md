Writing, executing, and debugging browser tests using Playwright.

## Core Concepts

- **Auto-Waiting**: Relying on built-in waiting rules to bypass random sleep parameters.
- **Trace Viewer**: Recording screencasts, console outputs, and network calls for failures.

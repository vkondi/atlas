# Accessibility Basics

Practices and standards ensuring web contents are navigable by all users.

## Core Concepts

- **WCAG Guidelines**: Principles governing perceivable, operable, understandable, and robust interfaces.
- **ARIA Attributes**: Enhancing plain element contexts for screen readers.

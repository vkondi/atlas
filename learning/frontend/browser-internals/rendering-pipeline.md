# The Rendering Pipeline

The steps a browser executes to compile HTML, CSS, and JS into visible screens.

## Core Concepts

- **Parsing & Tree Construction**: Creating DOM and CSSOM trees.
- **Reflow, Paint, and Composite**: Calculating styles, positioning elements, and painting layers.

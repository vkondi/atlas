# Factory Pattern

Delegating object construction procedures to dedicated instantiators.

## Core Concepts

- **Encapsulated Instantiation**: Decoupling instantiator logic from target application modules.
- **Polymorphic Output**: Returning diverse subclass objects based on inputs.

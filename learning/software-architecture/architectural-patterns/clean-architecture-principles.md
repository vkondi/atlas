# Clean & Hexagonal Architecture

Decoupling business logic from external frameworks, libraries, and databases.

## Core Concepts

- **Dependency Rule**: Dependencies point inward toward core business domain rules.
- **Ports & Adapters**: Abstracting database schemas and external services behind interfaces.

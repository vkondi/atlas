# Serverless Computing

Running event-triggered functions without dedicating permanent virtual server resources.

## Core Concepts

- **FaaS Runtimes**: AWS Lambda execution triggers and horizontal auto-scaling.
- **Cold Start Tuning**: Reducing initialization pauses via runtime choice and bundle shrinking.

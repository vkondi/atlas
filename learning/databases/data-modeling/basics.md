# Data Modeling Fundamentals

Mapping business domain properties to database storage layouts.

## Core Concepts

- **Normalization**: Reducing redundancy (1NF, 2NF, 3NF structures).
- **Denormalization**: Composing redundant fields to minimize query latency in write-heavy systems.

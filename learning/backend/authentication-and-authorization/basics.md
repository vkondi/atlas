# Authentication & Authorization

Securing API entry points, verifying identities, and validating access permissions.

## Core Concepts

- **AuthN vs AuthZ**: Verifying identity keys vs evaluating permissions.
- **Security Foundations**: Password hashing algorithms (argon2, bcrypt) and salt generation.

# Node.js Internals

Under the hood of the Node.js V8 execution engine and asynchronous I/O loops.

## Core Concepts

- **libuv**: Managing OS-level threadpools and kernel events.
- **Worker Threads**: Executing CPU-bound workloads without blocking the primary loop.
